# Space Document RAG System

> **Status: ⚪ Planned** — Portfolio Project 2.

A retrieval-augmented generation (RAG) system over real space-domain documents — NASA mission reports, ESA research papers, and Copernicus environmental data.

## What this project will demonstrate

- Document ingestion from public space-domain sources
- Embeddings and a vector store
- A retrieval pipeline with chunking and reranking
- A query interface
- **Retrieval-quality evaluation with RAGAS**, with documented results

See the [roadmap](../ROADMAP.md#-phase-3--rag--retrieval-systems-planned) for where this fits in the journey.
